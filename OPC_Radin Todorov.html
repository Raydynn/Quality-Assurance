Scope

The purpose of this project is to test the functionality of an online parking calculator. The application gives the possibility for users to choose different types of parking places and different periods to park. For each parking type and depending on the selected time frame, there will be different tax calculated.


1.Test Suite : Functional Testing

Testing of common features (setting date/time frame, Time frame calculation, Cost calculation, common error messages)
